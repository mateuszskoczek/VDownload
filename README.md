# VDownload


VDownload is an universal video downloader, which allows you to download videos from the following websites:
- Youtube (video, channel (P), playlist (P))
- Twitch (vod, clip, channel (P))
- ... [*more pages will be added later*]



## Setup


#### Requirements

- Operating system: Windows 7 +
- Architecture: x86/x64/arm64 (*ARM build have been not tested*)
- .NET Runtime 5.0 + [[LINK]](https://dotnet.microsoft.com/download)
- Microsoft Visual C++ 2015 Redistributable [[LINK]](https://www.microsoft.com/en-US/download/details.aspx?id=48145)
- Optional: FFmpeg (*If you have FFmpeg installed on your computer, you can change used installation of FFmpeg in settings and delete included FFmpeg folder*)

#### Installation

Executable is available [HERE](https://github.com/mateuszskoczek/VDownload/releases).

1. Download executable suitable for your architecture.
2. Put files in any folder
3. Add folder path to PATH (optional, but if you don't you will have to open terminal in app folder to use the app)

#### Update

1. Download executable suitable for your architecture.
2. Remove old files in installation folder
3. Put new files in installation folder
